# 📚 毎日の金融リテラシー学習ノート

Claude Code の `/finance-study` スキルで毎朝6時に自動生成される学習コンテンツです。

## 学習テーマ

| テーマ | 内容 |
|--------|------|
| 家計管理の基本 | 収支把握・予算設計・緊急予備資金 |
| 投資の基礎知識 | 株式・債券・投資信託・ETF・REIT |
| NISA・iDeCo | 節税投資制度の活用戦略 |
| 日本の税金 | 所得税・住民税・確定申告・控除 |
| 保険の選び方 | 生命・医療・火災保険の考え方 |
| 不動産・住宅ローン | 賃貸 vs 購入、ローンのリスク |
| 老後のお金 | 年金・退職金・4%ルール |
| リスク管理 | ポートフォリオ設計・リバランス |
| 経済の読み方 | 金利・為替・インフレ・景気 |
| 資産形成戦略 | ドルコスト平均法・複利・長期投資 |

## ディレクトリ構成

```
finance-study-journal/
└── YYYY/
    └── MM/
        └── YYYY-MM-DD.md  ← 毎日自動追加
```

## 自動化について

- **実行時刻**: 毎朝 6:00 (macOS launchd)
- **生成**: Claude Code CLI (`claude -p "/finance-study"`)
- **保存**: GitHub に自動コミット・プッシュ

> ⚠️ 本ノートは学習目的であり、投資アドバイスではありません。
> 実際の投資判断は自己責任で行ってください。
