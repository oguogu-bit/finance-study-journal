

### 📅 今日の学習テーマ：暗号資産・オルタナティブ投資 — ビットコインのボラティリティと資産配分の考え方

**難易度：** ⭐⭐⭐（応用）

### 📖 今日の解説（5〜10分で読める量）

#### そもそも「オルタナティブ投資」とは？

オルタナティブ（alternative）とは「代替的な」という意味です。株式や債券といった**伝統的資産（トラディショナル資産）**に対して、それ以外の投資対象を総称してオルタナティブ投資と呼びます。具体的には以下のようなものが含まれます。

| カテゴリ | 具体例 |
|---|---|
| 暗号資産（仮想通貨） | ビットコイン（BTC）、イーサリアム（ETH）など |
| 貴金属 | 金（ゴールド）、銀、プラチナ |
| コモディティ（商品） | 原油、天然ガス、農産物 |
| 不動産（実物） | 現物不動産、不動産ファンド |
| インフラ | インフラファンド、再生可能エネルギー |
| その他 | ヘッジファンド、プライベートエクイティ、アート、ワインなど |

今日はこの中でも特に**ビットコイン**に焦点を当て、「ポートフォリオに組み込むとしたらどう考えるか？」を学んでいきます。

---

#### ビットコインのボラティリティはどれくらい大きいのか？

**ボラティリティ（volatility）**とは、価格変動の大きさを示す指標です。標準偏差（σ）で表されることが多く、この数値が大きいほど「値動きが激しい＝リスクが高い」ことを意味します。

主要資産クラスの年率ボラティリティ（概算・過去10年平均）を比較してみましょう。

| 資産クラス | 年率ボラティリティ（目安） |
|---|---|
| 日本国債（10年） | 約2〜4% |
| 先進国株式（MSCI World） | 約15〜18% |
| 日本株式（TOPIX） | 約16〜20% |
| 金（ゴールド） | 約15〜20% |
| **ビットコイン** | **約50〜80%** |

ビットコインは株式の**3〜5倍**のボラティリティを持つことがわかります。これは、100万円分のビットコインを保有していた場合、1年後に50万円になることも、200万円になることも十分あり得るということです。

実際に過去の大幅下落を見てみましょう。

| 時期 | 下落率（ピークからの最大下落） |
|---|---|
| 2018年（バブル崩壊） | 約−84%（約220万円→約35万円） |
| 2022年（テラ/FTX危機） | 約−77%（約750万円→約170万円） |

株式市場のリーマンショック時の下落（S&P500で約−57%）と比べても、ビットコインの下落幅はさらに大きいことが分かります。

#### なぜそれでもポートフォリオに入れる人がいるのか？

ビットコインがここまでリスクが高いにもかかわらず、資産配分に組み込む投資家がいる理由は主に3つです。

**① 長期リターンの高さ**

ビットコインは2009年の誕生以来、暴落を何度も経験しながらも、長期的には非常に高いリターンを記録してきました。もちろん「過去のリターンは将来を保証しない」ことは大前提ですが、長期保有者の多くが大きな利益を得てきたのも事実です。

**② 他の資産クラスとの相関が比較的低い**

**相関係数**とは、2つの資産が同じ方向に動くかを示す指標で、−1〜+1の範囲で表されます。

- **+1に近い** → 同じ方向に動く（分散効果が薄い）
- **0に近い** → 無関係に動く（分散効果が期待できる）
- **−1に近い** → 逆方向に動く（強い分散効果）

ビットコインと主要資産の相関係数（近年の概算）は以下のとおりです。

| 組み合わせ | 相関係数（目安） |
|---|---|
| ビットコイン × S&P500 | 約0.3〜0.5 |
| ビットコイン × 金 | 約0.1〜0.3 |
| ビットコイン × 日本国債 | 約0.0〜0.1 |

株式との相関は近年やや高まっているものの、金や債券とはほぼ無相関です。ポートフォリオ全体で見ると、**少量の配分で分散効果を得られる可能性**があります。

**③ 「デジタルゴールド」としてのインフレヘッジ期待**

ビットコインは発行上限が**2,100万枚**と決まっています。法定通貨のように中央銀行が増刷することはできないため、金と同様に「希少性のある価値保存手段」としてインフレヘッジの役割を期待する見方があります。ただし、この点は学術的にはまだ議論が続いています。

#### 資産配分に組み込む場合の「上限」を考える

ここが今日の最も重要なポイントです。ビットコインのような超高ボラティリティ資産をポートフォリオに入れる際は、**全体に与えるリスク寄与度**を基準に考えます。

一般的に、金融の研究や機関投資家のレポートで提案されている配分比率は以下のとおりです。

| 配分比率 | 評価 |
|---|---|
| 0%（持たない） | 最も保守的。分からないものには投資しないのも立派な判断 |
| 1〜3% | 少量配分。ポートフォリオ全体のリスクを大幅に増やさずにリターン向上の可能性 |
| 5% | 積極配分。ビットコインに対する強い確信がある場合 |
| 10%以上 | 高リスク。暴落時にポートフォリオ全体が大きく傷む可能性 |

**重要な考え方は「最悪ゼロになっても生活に影響しない金額」を上限にすること**です。暗号資産には株式や債券と異なり、発行体の裏付けとなる企業の利益やキャッシュフローがありません。規制変更や技術的リスク（ハッキング等）で価値が大きく毀損する可能性もゼロではありません。

#### 日本における暗号資産の税制（2025年度時点）

暗号資産投資を検討するうえで、**税金の扱い**は必ず理解しておくべきポイントです。

| 項目 | 内容 |
|---|---|
| 所得区分 | **雑所得**（総合課税） |
| 税率 | 所得に応じて**住民税10%+所得税5〜45%**＝最大約55% |
| 損益通算 | 株式や投資信託の利益とは**通算不可** |
| 損失繰越 | **不可**（翌年以降に繰り越せない） |
| NISAの対象 | **対象外** |

株式投資の場合は申告分離課税で一律**約20.315%**ですが、暗号資産の場合は総合課税のため、利益が大きくなるほど税率が上がります。年収400万円の会社員が暗号資産で100万円の利益を得た場合、所得税・住民税合わせて**約30%（約30万円）**程度の税金がかかる計算になります。

この税制上の不利は、暗号資産の配分比率を考える上で無視できない重要な要素です。

### 💡 今日のキーポイント（5項目）

- **ビットコインのボラティリティは株式の3〜5倍。** 短期間で資産が半減することも珍しくないため、投資額は「最悪ゼロになっても問題ない範囲」に限定する
- **ポートフォリオへの配分は1〜5%が目安。** 少量でも分散効果が期待できるが、10%を超えると全体のリスクが大幅に上昇する
- **相関係数に注目。** ビットコインは金や債券との相関が低く、ポートフォリオ全体のリスク・リターン改善に寄与する可能性がある
- **日本の税制では暗号資産は「雑所得」扱い。** 最大約55%の税率、損失繰越不可、NISAの対象外と、株式投資に比べて税制面で大きく不利
- **「理解できないものには投資しない」も賢明な判断。** ビットコインの技術やリスクを十分理解してから投資を検討しても遅くない

### 🧮 実践例・計算例

#### 【シミュレーション】ビットコインを3%組み入れた場合のポートフォリオ比較

**前提条件（26歳・年収400万円・投資資金100万円）**

| | ポートフォリオA（伝統的） | ポートフォリオB（BTC混合） |
|---|---|---|
| 全世界株式 | 80万円（80%） | 77万円（77%） |
| 先進国債券 | 20万円（20%） | 20万円（20%） |
| ビットコイン | 0円（0%） | 3万円（3%） |

**年率リターン・リスクの期待値（概算）**

| | ポートフォリオA | ポートフォリオB |
|---|---|---|
| 期待リターン（年率） | 約5.5% | 約6.5% |
| ボラティリティ（年率） | 約13% | 約14.5% |
| シャープレシオ（目安） | 約0.42 | 約0.45 |

> **シャープレシオ** = （リターン − 無リスク金利）÷ ボラティリティ。この値が高いほど「リスクに対するリターン効率が良い」ことを意味します。

ポートフォリオBはリスクがやや増えますが、リターン向上のほうが大きいため、シャープレシオが若干改善しています。

#### 【最悪シナリオの計算】ビットコインが80%暴落した場合

- ポートフォリオA：株式−30%として → 100万円 → **約76万円（−24%）**
- ポートフォリオB：株式−30%＋BTC−80%として → 100万円 → **約74.6万円（−25.4%）**

差はわずか**1.4万円（1.4%ポイント）**。3%の配分であれば、ビットコインが壊滅的な下落をしても全体への影響は限定的です。

#### 【税金シミュレーション】BTC利益が出た場合（年収400万円の場合）

3万円分のビットコインが10倍になり30万円に → 利益27万円

| 項目 | 金額 |
|---|---|
| 暗号資産の利益 | 27万円 |
| 所得税率（課税所得450万円相当） | 約20%（控除あり） |
| 住民税 | 10% |
| 概算税額 | 約27万円 × 30% = **約8.1万円** |
| 手取り利益 | **約18.9万円** |

株式で同じ利益なら税金は約5.5万円（20.315%）なので、暗号資産のほうが**約2.6万円多く**税金を取られる計算です。

### ✅ 今日のアクションアイテム

1. **自分のポートフォリオ全体を書き出し、暗号資産の比率を確認する**
現在の保有資産（NISAの投資信託、預貯金、暗号資産など）を一覧にして、暗号資産が全体の何%を占めているか計算してみましょう。5%を超えている場合はリスクの再確認を。

2. **暗号資産の利益が出ている場合、確定申告の準備を始める**
暗号資産の利益は年間20万円を超えると確定申告が必要です（会社員の場合）。取引履歴をダウンロードし、損益計算ツール（CryptactやGtaxなど）で把握しておきましょう。

3. **「最悪ゼロになってもいい金額」を具体的に決める**
感覚ではなく具体的な金額を紙に書き出してください。例えば貯蓄120万円のうち緊急予備資金75万円を除いた投資可能額45万円の3%＝約1.3万円が上限目安です。

### ❓ 理解度チェッククイズ（3問）

**Q1. ビットコインの年率ボラティリティは、株式（先進国）のおよそ何倍ですか？**

> **A1. 約3〜5倍**
> 先進国株式の年率ボラティリティが約15〜18%に対し、ビットコインは約50〜80%と、概ね3〜5倍の変動幅があります。

---

**Q2. 日本において、暗号資産の売却益はどの所得区分に分類され、株式と比べた税制上の不利な点は何ですか？**

> **A2. 「雑所得」（総合課税）に分類される。** 株式の申告分離課税（一律約20.315%）と異なり、他の所得と合算されて累進課税（最大約55%）が適用される。さらに、損失繰越ができず、NISAの対象外である点も不利。

---

**Q3. ポートフォリオにビットコインを組み入れる際、一般的に推奨される配分比率の目安は何%で、その理由は何ですか？**

> **A3. 1〜5%が目安。** この程度の配分であれば、ビットコインが暴落してもポートフォリオ全体への影響は限定的でありながら、他の資産との低い相関を活かした分散効果とリターン向上が期待できるため。
