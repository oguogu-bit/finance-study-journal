

# 📅 今日の学習テーマ：不動産と住宅ローン

**日付：** 2026年3月5日（木）｜テーマ6/10

**難易度：** ⭐⭐（基礎）

**サブトピック：** 賃貸 vs 購入、26歳の今どう考えるべきか？

---

## 📖 今日の解説

### 「家は買うべき？借り続けるべき？」——永遠の論争に答えを出す考え方

26歳、一人暮らし。周りで「そろそろ家を買おうかな」という声が聞こえ始める時期かもしれません。でも、焦る必要はありません。大切なのは **「どちらが得か」ではなく「自分の人生設計に合っているか」** という視点です。

---

### 🏠 賃貸のメリット・デメリット

#### メリット
- **柔軟性が高い**：転職・転勤・結婚など、ライフステージの変化に対応しやすい
- **初期費用が少ない**：敷金・礼金・仲介手数料で家賃の4〜5ヶ月分程度
- **修繕費用がかからない**：設備の故障や建物の老朽化は大家さん負担
- **固定資産税がかからない**：不動産を所有していないため課税されない

#### デメリット
- **資産が残らない**：毎月の家賃は「消費」であり、支払っても自分のものにならない
- **家賃は一生続く**：老後も家賃を払い続ける必要がある
- **自由度が低い**：リフォームや大きな模様替えが制限される
- **高齢になると借りにくくなる可能性**：保証人の問題や大家の審査が厳しくなることも

---

### 🏡 購入のメリット・デメリット

#### メリット
- **資産になる**：ローン完済後は自分の財産（ただし価値は変動する）
- **住居費が将来的に下がる**：ローン完済後は固定資産税・管理費のみ
- **住宅ローン控除が使える**：最大13年間、年末ローン残高の0.7%が所得税等から控除
- **団体信用生命保険（団信）**：万一死亡した場合、ローン残高がゼロになる

#### デメリット
- **初期費用が大きい**：物件価格の5〜10%が諸費用としてかかる
- **流動性が低い**：売りたいときにすぐ売れるとは限らない
- **維持費がかかる**：固定資産税（年間10〜15万円程度）、修繕費、管理費
- **価格下落リスク**：購入した価格より下がる可能性がある
- **金利変動リスク**：変動金利の場合、将来の返済額が増える可能性

---

### 🔑 住宅ローンの基本用語

| 用語 | 意味 |
|------|------|
| **元利均等返済** | 毎月の返済額（元金＋利息）が一定。最初は利息の割合が大きい |
| **元金均等返済** | 毎月の元金返済額が一定。最初は返済額が大きく、徐々に減る |
| **固定金利** | 借入期間中ずっと同じ金利。返済計画が立てやすいが、変動金利より高め |
| **変動金利** | 半年ごとに金利が見直される。当初は低いが、将来上がるリスクがある |
| **団体信用生命保険（団信）** | ローン契約者が死亡・高度障害になった場合、残債がゼロになる保険 |
| **頭金** | 物件価格のうち、ローンを使わず自己資金で支払う部分 |
| **返済比率** | 年収に対する年間返済額の割合。一般に**25%以内**が安全圏 |

### 📊 2024〜2025年の住宅ローン金利の目安

| 金利タイプ | 金利の目安 |
|-----------|-----------|
| 変動金利 | 年0.3〜0.6%程度 |
| 固定10年 | 年1.0〜1.5%程度 |
| 全期間固定（フラット35） | 年1.5〜2.0%程度 |

> ⚠️ **注意：** 2024年以降、日銀の利上げにより変動金利も上昇傾向にあります。「変動金利は安いから」と安易に選ぶのではなく、将来の金利上昇も想定して判断しましょう。

### 🧭 26歳の今、どう考えるべきか？

**結論：今すぐ急いで買う必要はありません。**

理由は以下の通りです：

1. **ライフプランが未確定**：転職・結婚・転勤など、今後数年で大きく状況が変わる可能性が高い
2. **頭金が十分でない**：貯蓄100〜150万円では、購入の諸費用で消えてしまう
3. **まずは「買える力」を蓄える時期**：頭金を貯めながら、住宅ローンや不動産の知識を身につけることが最善の投資

**今やるべきことは、「賃貸で柔軟性を確保しつつ、将来の選択肢を増やすための貯蓄と学習」です。**

---

## 💡 今日のキーポイント

- **「賃貸＝損」「購入＝得」は単純な間違い。** ライフプランと経済状況によって最適解は変わる
- **住宅ローンの返済比率は手取りの25%以内が目安。** 無理なローンは人生最大のリスクになり得る
- **変動金利の安さだけで判断しない。** 金利上昇リスクを必ずシミュレーションに含める
- **住宅購入の「隠れコスト」を忘れない。** 諸費用・固定資産税・修繕費・管理費は購入後も続く
- **26歳の最優先は「選択肢を広げる力」を蓄えること。** 焦って買うより、知識と資金を貯める方が合理的

## 🧮 実践例・計算例

### ケース：26歳・年収400万円（手取り月25万円）が「もし今、家を買ったら？」

#### 前提条件
- 物件価格：2,500万円（単身向け1LDKマンション想定）
- 頭金：100万円
- 借入額：2,400万円
- 金利：変動0.5%（当初）
- 返済期間：35年
- ボーナス返済：なし

#### 月々の返済額

```
【変動金利 0.5%の場合】
  月々の返済額：約 62,300円
  年間返済額 ：約 747,600円
  返済比率   ：約 18.7%（年収400万円に対して）→ ✅ 安全圏

【もし金利が2.0%に上がったら】
  月々の返済額：約 79,700円（＋17,400円/月の増加！）
  年間返済額 ：約 956,400円
  返済比率   ：約 23.9% → ⚠️ ギリギリ

【もし金利が3.0%に上がったら】
  月々の返済額：約 92,300円（＋30,000円/月の増加！）
  年間返済額 ：約 1,107,600円
  返済比率   ：約 27.7% → ❌ 危険ゾーン
```

#### 購入時の諸費用（物件価格の約7%）

| 項目 | 金額の目安 |
|------|-----------|
| 仲介手数料 | 約89万円 |
| 登記費用 | 約30万円 |
| ローン事務手数料 | 約50万円 |
| 火災保険（10年） | 約15万円 |
| 不動産取得税 | 約10万円 |
| その他（印紙税等） | 約6万円 |
| **合計** | **約200万円** |

> 💸 頭金100万円＋諸費用200万円＝**最低でも300万円の自己資金が必要。** 現在の貯蓄100〜150万円では大幅に足りません。

#### 月々の維持費（購入後にかかるお金）

| 項目 | 月額の目安 |
|------|-----------|
| 管理費 | 約12,000円 |
| 修繕積立金 | 約8,000円 |
| 固定資産税（月割り） | 約8,000円 |
| **合計** | **約28,000円** |

```
ローン返済 62,300円 ＋ 維持費 28,000円 ＝ 月々約 90,300円

→ 手取り25万円に対して 約36% が住居費に消える！
```

#### 比較：同じエリアの賃貸の場合

```
家賃 65,000円（1LDK・都市近郊の想定）

→ 手取り25万円に対して 約26% が住居費

差額：90,300円 − 65,000円 ＝ 月 25,300円

この差額を年利5%で30年間積立投資したら…
→ 約 2,100万円 になる可能性！
```

> ⚡ つまり「購入して資産を作る」のと「賃貸で浮いた分を投資に回す」のは、条件次第でどちらが有利になるか変わります。**単純に「家賃がもったいない」とは言えないのです。**

## ✅ 今日のアクションアイテム

### 1. 自分の「住居費比率」を計算してみよう
```
現在の家賃 ÷ 手取り月収 × 100 ＝ ○○%
```
目安：**25〜30%以内**なら健全。30%を超えていたら、家賃の見直しを検討。

### 2. 住宅ローンシミュレーターで遊んでみよう
金融機関のWebサイトにある無料シミュレーターで、物件価格・金利・返済期間を変えてみましょう。「金利が1%上がるとどうなるか」を体感することが大切です。

### 3.「住宅購入は○歳までに△万円貯める」というマイルストーンを仮置きしよう
例えば「32歳までに500万円（頭金＋諸費用）を貯める」と決めれば、毎月の貯蓄目標が明確になります。
```
500万円 ÷ 6年（72ヶ月）＝ 月約 69,500円
```
> 現実的に厳しければ、目標額や時期を調整してOK。大事なのは**数字で考える習慣**です。

## ❓ 理解度チェッククイズ

### Q1. 住宅ローンの「返済比率」は年収の何%以内が安全圏とされていますか？

<details>
<summary>答えを見る</summary>

**25%以内**が一般的な安全圏の目安です。金融機関の審査基準では30〜35%まで借りられることもありますが、「借りられる額」と「返せる額」は違います。余裕を持った返済計画が大切です。

</details>

---

### Q2. 住宅ローンの「団体信用生命保険（団信）」とは何ですか？

<details>
<summary>答えを見る</summary>

住宅ローンの契約者が**死亡または高度障害状態**になった場合に、**残りのローン残高がゼロになる**（保険で完済される）保険のことです。多くの住宅ローンでは加入が必須で、保険料はローン金利に含まれていることが一般的です。

</details>

---

### Q3. 2,500万円のマンションを購入する場合、物件価格以外に必要な「諸費用」はおよそいくらですか？

<details>
<summary>答えを見る</summary>

物件価格の約**5〜10%**、つまり**約125万〜250万円**が目安です。仲介手数料・登記費用・ローン事務手数料・火災保険・不動産取得税などが含まれます。「頭金ゼロで買える！」と言われても、諸費用分の現金は別途必要です。

</details>
