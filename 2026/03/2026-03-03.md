## 📖 今日の解説

### 「賃貸 vs 購入」— 26歳の今、知っておくべき住まいのお金の真実

住まいは人生最大の支出の一つです。「家賃を払い続けるのはもったいない」「持ち家は資産になる」——こんな言葉を聞いたことがあるかもしれません。しかし、本当にそうでしょうか？ 今日は**冷静にお金の面から住まいを考える力**を身につけましょう。

---

### 1. 賃貸と購入、それぞれのコスト構造

#### 🏠 賃貸のコスト

| 項目 | 内容 |
|------|------|
| **家賃** | 毎月の支払い（東京23区1K：約7〜9万円） |
| **管理費・共益費** | 月3,000〜5,000円程度 |
| **更新料** | 2年ごとに家賃1ヶ月分（地域による） |
| **初期費用** | 敷金・礼金・仲介手数料（家賃の3〜5ヶ月分） |
| **火災保険** | 年1〜2万円程度 |

> **ポイント：** 賃貸は「使用料」を払っている状態。資産は残らないが、身軽さ・柔軟性がある。

#### 🏡 購入のコスト

| 項目 | 内容 |
|------|------|
| **住宅ローン返済** | 元本＋利息（後述で詳しく解説） |
| **頭金** | 物件価格の10〜20%が目安 |
| **諸費用** | 物件価格の6〜10%（登記費用・仲介手数料・ローン手数料等） |
| **固定資産税** | 年間10〜15万円程度（物件による） |
| **管理費・修繕積立金** | マンションの場合、月2〜4万円 |
| **修繕費用** | 戸建ての場合、10〜15年ごとに100〜200万円 |
| **火災保険・地震保険** | 年2〜5万円程度 |

> **ポイント：** 購入は「資産を持つ」が、ローン返済が終わるまでは銀行のもの。**見えないコスト（税金・修繕・保険）が多い**ことを理解しよう。

---

### 2. 住宅ローンの基本的な仕組み

住宅ローンとは、住宅を購入するために金融機関からお金を借りる仕組みです。

#### 金利タイプ

| タイプ | 特徴 | 2024年時点の目安 |
|--------|------|-----------------|
| **変動金利** | 半年ごとに金利が見直される。当初は低いが将来上がるリスクあり | 年0.3〜0.6%程度 |
| **固定金利（全期間）** | 借入時の金利が完済まで変わらない。安心だがやや高め | 年1.5〜2.0%程度 |
| **固定期間選択型** | 最初の5年・10年だけ固定、その後変動に切り替わる | 年0.8〜1.5%程度 |

> **用語解説**
> - **元利均等返済**：毎月の返済額（元本＋利息）が一定になる方式。返済初期は利息の割合が大きい。
> - **元金均等返済**：毎月返済する元本が一定で、利息分が徐々に減る方式。最初の負担は大きいが総支払額は少ない。

#### 住宅ローン控除（住宅借入金等特別控除）

住宅ローンを組むと、税金が戻ってくる制度があります。

- **控除額：** 年末ローン残高の **0.7%** が所得税（＋住民税の一部）から控除
- **控除期間：** 新築は最長 **13年間**、中古は **10年間**
- **借入限度額：** 新築（省エネ基準適合）で **3,000〜5,000万円**（住宅の環境性能による）
- **所得要件：** 合計所得金額 **2,000万円以下**

### 3. 26歳で「買うべきか？」を考えるフレームワーク

結論を急ぐ必要はありません。以下の観点で整理しましょう。

| 判断基準 | 賃貸が有利なケース | 購入が有利なケース |
|----------|-------------------|-------------------|
| **転勤・転職の可能性** | 高い → 賃貸 | 低い（同じ地域に10年以上住む見込み） |
| **ライフプランの確度** | 結婚・家族構成が未定 | ある程度確定している |
| **頭金・貯蓄** | まだ十分でない | 物件価格の20%＋諸費用を確保できる |
| **住みたいエリアの相場** | 家賃が割安な地域 | 購入の方が月々安くなる地域 |
| **資産性** | こだわらない | 将来売却・賃貸に出す想定がある |

> 💬 **26歳・独身・年収400万円**の現時点では、**多くの場合「賃貸を続けながら資産形成に集中する」のが合理的**です。住宅購入は頭金が貯まり、ライフプランが見えてきてからでも遅くありません。

---

## 💡 今日のキーポイント

1. **「家賃がもったいない」は半分ウソ** — 購入には固定資産税・修繕費・管理費・保険料など「見えないコスト」が多く、家賃との単純比較はできない

2. **住宅ローンの金利タイプは人生を左右する** — 変動金利は安いが金利上昇リスクがある。「今の低金利が35年続く」前提で借りるのは危険

3. **住宅購入の目安は「年収の5倍以内」** — 無理のない返済は手取りの25%以内。年収400万円なら借入額2,000万円以内が安全圏

4. **「買う vs 借りる」に正解はない** — ライフプラン・住むエリア・金利環境によって最適解は変わる。大事なのは**数字で比較する力**

5. **26歳の最優先事項は「選択肢を増やすこと」** — 頭金を貯めながらNISA等で資産形成し、いつでも買える状態を作ることが最善

## 🧮 実践例・計算例

### 【ケース】26歳・年収400万円が3,000万円のマンションを買ったら？

#### 前提条件
- 物件価格：3,000万円（都内1LDK中古マンション）
- 頭金：300万円（貯蓄150万円のうち使えるのは…→ **頭金すら不足！**）
- 借入額：2,700万円
- 金利：変動0.5%（当初）
- 返済期間：35年（元利均等返済）

#### 月々の返済シミュレーション

```
【ローン返済のみ】
  借入額 2,700万円 ÷ 35年 ＋ 利息
  → 月々約 70,000円

【実際の月々の住居費】
  ローン返済：      70,000円
  管理費・修繕積立： 25,000円
  固定資産税（月割）：10,000円
  火災・地震保険：    3,000円
  ─────────────────
  合計：約 108,000円／月
```

#### 現在の賃貸と比較

```
【現在の賃貸（1K・家賃7.5万円の場合）】
  家賃：            75,000円
  管理費：           3,000円
  ─────────────────
  合計：約 78,000円／月

  → 購入すると月々の住居費が約3万円アップ！
```

#### 35年間の総支払額比較（概算）

```
【購入の場合】
  頭金：              300万円
  諸費用（物件の7%）：  210万円
  ローン総返済額：    2,940万円（利息約240万円）
  管理費等（35年）：  1,050万円
  固定資産税（35年）：  420万円
  修繕費用：           200万円
  ─────────────────────
  総支出：約 5,120万円
  ※35年後に物件（資産）が残る（ただし築年数により価値は大幅下落の可能性）

【賃貸の場合（家賃据え置きで計算）】
  家賃＋管理費（35年）：3,276万円
  更新料（17回分）：    127万円
  ─────────────────────
  総支出：約 3,403万円
  ※資産は残らないが、差額1,717万円を投資に回せる
```

#### 💰 差額を投資した場合

```
  月々の差額：約3万円をNISAで積立投資（年利5%想定）
  → 35年後：約 3,400万円（元本1,260万円＋運用益約2,140万円）

  ※投資には元本割れリスクがあり、年利5%を保証するものではありません
```

> 📌 **この計算が示すこと：** 「賃貸＋投資」と「住宅購入」は、条件次第でどちらも合理的になり得る。重要なのは**感覚ではなく数字で判断する**こと。

## ✅ 今日のアクションアイテム

### 1. 自分の「住居費比率」を計算してみよう
```
現在の住居費（家賃＋管理費）÷ 手取り月収 × 100 ＝ ？%

例：78,000円 ÷ 250,000円 × 100 ＝ 31.2%

→ 目安は25%以内が理想。30%超えなら家賃の見直しも検討
```

### 2. 住宅ローンシミュレーターで遊んでみよう
金融機関のWebサイトにある無料シミュレーターで、いろんな条件（金利・借入額・期間）を入力して、返済額の変化を体感してみましょう。「もし金利が2%に上がったら？」も必ず試してみてください。

### 3. 「購入に必要な貯蓄額」を逆算しよう
```
仮に3,000万円の物件を買うなら：
  頭金（20%）：     600万円
  諸費用（7%）：    210万円
  引越し費用等：      30万円
  緊急予備資金（残す分）：100万円
  ─────────────────
  必要貯蓄：約 940万円

  → 今の貯蓄150万円からあと約800万円。月5万円貯蓄で約13年後（39歳）
```

### Q2. 住宅購入時、物件価格以外にかかる「諸費用」はおよそ何%が目安？

<details>
<summary>答えを見る</summary>

**物件価格の約6〜10%**（中古の場合は仲介手数料が加わるため高くなる傾向）

内訳例：登記費用、住宅ローン手数料・保証料、仲介手数料、火災保険料、印紙税など。3,000万円の物件なら180〜300万円程度の諸費用がかかります。
</details>

---

### Q3. 年収400万円の場合、無理のない住宅ローン借入額の目安はいくら？

<details>
<summary>答えを見る</summary>

**約2,000万円以内（年収の5倍以内）**

金融機関の審査上は年収の7〜8倍まで借りられることもありますが、**「借りられる額」と「返せる額」は違います**。手取りの25%以内に月々の返済を抑えることが、生活を圧迫しない目安です。
</details>
