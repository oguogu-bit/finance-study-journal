

# 📅 今日の学習テーマ：不動産と住宅ローン

**日付：** 2026年3月5日（木）｜ テーマ6/10

**難易度：** ⭐⭐（基礎）

**今日のサブトピック：** 賃貸 vs 購入、本当に「家賃はもったいない」のか？

---

## 📖 今日の解説

### 「家賃を払い続けるのはもったいない」は本当か？

26歳の会社員として、周りから「そろそろ家を買ったら？」「家賃を払い続けるのはもったいないよ」と言われることがあるかもしれません。しかし、この言葉を鵜呑みにする前に、**賃貸と購入それぞれのコスト構造**をしっかり理解することが大切です。

---

### 🏠 賃貸のコスト構造

賃貸で支払うお金は、家賃だけではありません。

| 項目 | 内容 |
|------|------|
| **家賃** | 毎月の住居費（例：月7万円） |
| **管理費・共益費** | 共用部分の維持費（例：月5,000円） |
| **更新料** | 通常2年ごとに家賃1ヶ月分 |
| **火災保険料** | 年1〜2万円程度 |
| **引っ越し費用** | 転居のたびに発生（敷金・礼金・仲介手数料など） |

**賃貸の特徴：**
- ✅ ライフステージに合わせて**柔軟に住み替え**できる
- ✅ 設備の故障・修繕は**大家さん負担**
- ✅ 固定資産税・修繕積立金が**不要**
- ❌ いくら払っても**資産にはならない**
- ❌ 高齢になると**契約を断られる**リスクがある

---

### 🏡 購入のコスト構造

「家を買う＝住宅ローンの返済額だけ」と考えるのは大きな間違いです。購入には**見えにくいコスト**が多数あります。

| 項目 | 内容 |
|------|------|
| **住宅ローン返済** | 元本＋利息（例：月8〜10万円） |
| **住宅ローンの利息総額** | 35年ローンだと物件価格の15〜30%に及ぶことも |
| **固定資産税・都市計画税** | 年間10〜20万円程度（物件による） |
| **修繕費・管理費** | マンションなら月2〜3万円、戸建ては自分で積立 |
| **購入時の諸費用** | 物件価格の**6〜10%**（登記費用、仲介手数料、ローン手数料等） |
| **火災保険・地震保険** | 年数万円〜十数万円 |

**購入の特徴：**
- ✅ ローン完済後は住居費が**大幅に減る**
- ✅ 資産として**残る**（ただし価値が下がる可能性あり）
- ✅ 住宅ローン控除で**税金の還付**を受けられる
- ❌ 簡単に**引っ越しできない**
- ❌ 金利上昇・収入減少時の**返済リスク**
- ❌ 建物の**経年劣化**と資産価値の下落

---

### 📊 住宅ローンの基本用語

| 用語 | 意味 |
|------|------|
| **元利均等返済** | 毎月の返済額が一定。初期は利息の割合が大きく、徐々に元本の割合が増える方式 |
| **元金均等返済** | 毎月の元本返済額が一定。初期の返済額が大きいが、総支払利息は少ない方式 |
| **固定金利** | 借入期間中の金利が変わらない（例：フラット35）。返済計画が立てやすい |
| **変動金利** | 半年ごとに金利が見直される。現在は低金利だが将来上がるリスクがある |
| **返済負担率** | 年収に対する年間ローン返済額の割合。**25%以下が安全圏**とされる |
| **団体信用生命保険（団信）** | ローン契約者が死亡・高度障害になった場合にローン残債がゼロになる保険 |
| **住宅ローン控除** | ローン残高の**0.7%**が最大13年間、所得税・住民税から控除される制度（2024年度） |

---

### 🤔 26歳で購入を急ぐ必要はない理由

1. **キャリアの変動リスク：** 転職・転勤・独立など、今後の人生は大きく変わりうる
2. **ライフプランの不確定性：** 結婚・出産により必要な間取りが変わる可能性
3. **頭金の準備：** 物件価格の10〜20%＋諸費用を貯めることで、ローンの負担を大きく減らせる
4. **金融リテラシーの向上：** 人生最大の買い物をする前に、投資・税金・保険の知識を深める時間がある

> **重要な考え方：** 「賃貸＝損、購入＝得」ではありません。**自分のライフプランに合った選択**が正解です。

## 💡 今日のキーポイント

- 📌 **家賃がもったいない＝購入が得、ではない。** 購入には利息・税金・修繕費など「見えないコスト」が大量に存在する
- 📌 **返済負担率は年収の25%以下に抑える。** これを超えると生活が圧迫され、投資に回す余裕もなくなる
- 📌 **変動金利の低さに飛びつかない。** 現在の低金利は歴史的に異例であり、将来の金利上昇リスクを織り込んで判断する
- 📌 **住宅は「投資」ではなく「消費」と考える。** 日本では多くの物件が経年で資産価値が下落する（特に戸建て）
- 📌 **26歳の今は「賃貸＋資産運用」が合理的な選択肢。** 頭金を貯めながらNISA等で運用し、ライフプランが固まってから検討しても遅くない

## 🧮 実践例・計算例

### 【ケース】26歳・年収400万円の場合を比較する

#### 🔸 パターンA：賃貸を続ける場合（月7万円）

```
家賃：         7.0万円/月
管理費・共益費： 0.5万円/月
──────────────────────
月額合計：      7.5万円

年間コスト：    7.5万円 × 12ヶ月 ＝ 90万円
更新料（2年毎）：7万円 → 年あたり3.5万円

▶ 年間の住居費合計：約93.5万円
▶ 10年間の住居費合計：約935万円
```

**浮いたお金を投資に回した場合：**
仮に毎月2万円をNISAで積立投資（年利5%想定）すると…

```
10年後の運用資産：約310万円（元本240万円＋運用益約70万円）
```

---

#### 🔸 パターンB：3,000万円のマンションを購入する場合

```
【購入時】
物件価格：       3,000万円
頭金（10%）：      300万円
諸費用（7%）：     210万円
──────────────────────
自己資金の合計：   510万円  ← 26歳で用意するのは現実的に厳しい

【住宅ローン】
借入額：        2,700万円
金利：          1.5%（固定35年）
毎月返済額：     約8.3万円（元利均等返済）
返済総額：       約3,480万円（利息だけで約780万円！）

【毎月のランニングコスト】
ローン返済：     8.3万円
管理費：        1.5万円
修繕積立金：     1.0万円
固定資産税（月割）：1.0万円
──────────────────────
月額合計：       約11.8万円

▶ 年間の住居費合計：約141.6万円
▶ 返済負担率：141.6万円 ÷ 400万円 ＝ 約35.4%  ⚠️ 危険水準！
```

---

#### 🔸 比較まとめ

| 項目 | 賃貸（パターンA） | 購入（パターンB） |
|------|:---:|:---:|
| 月額住居費 | 約7.5万円 | 約11.8万円 |
| 年間住居費 | 約93.5万円 | 約141.6万円 |
| 10年間の総コスト | 約935万円 | 約1,416万円＋頭金510万円 |
| 投資に回せる余力 | 月2〜3万円確保可能 | ほぼゼロ |
| 引越しの自由度 | ◎ | △ |
| 返済負担率 | — | 35.4%（⚠️超過） |
| 資産として残るもの | なし | マンション（ただし価値下落の可能性） |

> **結論：** 年収400万円で3,000万円の物件購入は返済負担率が25%を大きく超え、**かなり無理のある計画**。現段階では賃貸を続けながら頭金を貯め、年収アップやライフプランの確定を待つのが合理的。

## ✅ 今日のアクションアイテム

### 1. 自分の「住居費比率」を計算する
```
今の家賃（管理費込み）÷ 手取り月収 × 100 ＝ ？％
```
→ **30%以下**なら健全。超えていれば引っ越しや固定費の見直しを検討。

### 2. 「住宅購入に必要な自己資金」を逆算する
もし将来的に住宅購入を考えるなら、物件価格の**15〜20%（頭金10%＋諸費用5〜10%）**が目安。  
例：3,000万円の物件なら → 450〜600万円の自己資金が必要  
→ 毎月いくら貯めれば何年後に達成できるか計算してみよう。

### 3. 住宅ローンのシミュレーターを試してみる
金融機関や住宅金融支援機構の公式サイトにある無料シミュレーターで、借入額・金利・期間を入れて「返済総額」と「利息の大きさ」を体感する。

## ❓ 理解度チェッククイズ（3問）

---

**【Q1】** 住宅ローンの返済負担率は、年収の何%以下に抑えるのが望ましいとされていますか？

<details>
<summary>答えを見る</summary>

**25%以下**が安全圏とされています。  
例：年収400万円なら年間返済額100万円以下（月約8.3万円以下）が目安。これを超えると生活費や投資資金が圧迫されます。

</details>

---

**【Q2】** 住宅ローン控除（2024年度）では、ローン残高の何%が控除されますか？また、最大で何年間適用されますか？

<details>
<summary>答えを見る</summary>

ローン残高の**0.7%**が、最大**13年間**にわたって所得税・住民税から控除されます。  
例：ローン残高が2,700万円なら、年間最大18.9万円の税金が還付される計算です（所得税額が上限となる場合あり）。

</details>

---

**【Q3】** 3,000万円の住宅を購入する際、頭金10%＋諸費用7%とすると、購入時に用意すべき自己資金はいくらですか？

<details>
<summary>答えを見る</summary>

- 頭金：3,000万円 × 10% ＝ **300万円**
- 諸費用：3,000万円 × 7% ＝ **210万円**
- **合計：510万円**

諸費用には、登記費用・仲介手数料・ローン事務手数料・火災保険料・印紙税などが含まれます。「頭金ゼロでも買える」という広告を見ても、**諸費用は別途必ず現金で必要**になることを忘れないでください。

</details>
