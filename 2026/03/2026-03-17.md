## 📖 今日の解説

### そもそも「オルタナティブ投資」とは？

オルタナティブ（Alternative）投資とは、株式や債券といった**伝統的な資産クラス以外**への投資を指します。具体的には以下のようなものがあります。

| カテゴリ | 具体例 |
|---|---|
| 暗号資産（クリプト） | ビットコイン（BTC）、イーサリアム（ETH）など |
| 貴金属 | 金（ゴールド）、銀、プラチナ |
| コモディティ | 原油、天然ガス、農産物 |
| インフラ | インフラファンド、再生可能エネルギー施設 |
| その他 | ヘッジファンド、プライベートエクイティ、アート、ワインなど |

「お金に困らない人生」を設計するうえで、これらをポートフォリオに**少量加えるべきかどうか**が今日のテーマです。

### ビットコインの本質的な特性を理解する

ビットコインは2009年にサトシ・ナカモトの論文をもとに誕生した**分散型のデジタル通貨**です。投資対象として考える際に、まず以下の特性を押さえましょう。

#### 1. 供給量の上限がある

ビットコインの総発行量は**2,100万枚**と決められており、それ以上は絶対に増えません。これは金（ゴールド）と同様の「希少性」を持つという主張の根拠になっています。2024年4月には4度目の**半減期（マイニング報酬が半分になるイベント）**を迎え、新規供給ペースがさらに減少しました。

#### 2. ボラティリティ（価格変動幅）が極めて大きい

ここが最大のポイントです。「ボラティリティ」とは**価格がどれくらい上下に振れるかを示す指標**です。

| 資産 | 年率ボラティリティ（概算） |
|---|---|
| 日本国債 | 約2〜3% |
| 先進国株式（MSCIコクサイ） | 約15〜18% |
| S&P500 | 約15〜20% |
| 金（ゴールド） | 約15〜20% |
| ビットコイン | **約60〜80%** |

ビットコインのボラティリティは株式の**3〜5倍**にもなります。これは何を意味するかというと、たとえば100万円投資した場合、1年後に**40万円になることも、200万円を超えることも十分あり得る**ということです。

#### 3. 伝統資産との相関が不安定

オルタナティブ投資の魅力のひとつは「伝統資産と値動きが異なること（低相関）」ですが、ビットコインは**局面によって相関が変わる**という厄介な性質があります。

- **平常時**：株式との相関は比較的低い（0.1〜0.3程度）
- **金融危機・リスクオフ局面**：株式と一緒に暴落する傾向（相関が0.5〜0.8に急上昇）

つまり「株が下がったときに守ってくれる資産」とは限らないのです。2022年の暗号資産市場の暴落（FTX破綻など）では、ビットコインは高値から**約75%下落**しました。

### 資産配分にどう組み込むか？ — 「サテライト戦略」の考え方

資産運用の世界では、**コア・サテライト戦略**という考え方が広く使われています。

```
┌─────────────────────────────────────┐
│                                     │
│     【コア資産】80〜95%             │
│     ・全世界株式インデックス        │
│     ・先進国債券                    │
│     ・預金・個人向け国債            │
│         → 長期安定成長を担う        │
│                                     │
│  ┌─────────────────────┐            │
│  │【サテライト】5〜20% │            │
│  │ ・暗号資産           │            │
│  │ ・金                 │            │
│  │ ・個別株             │            │
│  │ ・REIT               │            │
│  │  → 超過リターン or   │            │
│  │    分散効果を狙う     │            │
│  └─────────────────────┘            │
└─────────────────────────────────────┘
```

**暗号資産はサテライト部分のさらに一部**として組み込むのが現実的です。多くのファイナンシャルアドバイザーが推奨する暗号資産の配分は、**ポートフォリオ全体の1〜5%以内**です。

#### なぜ5%以内なのか？

これはリスク管理の観点から逆算した数字です。仮にビットコインが**80%暴落**しても、ポートフォリオ全体に与える影響を見てみましょう。

| BTC配分比率 | BTCが80%下落した場合のポートフォリオ全体への影響 |
|---|---|
| 1% | −0.8%（ほぼ誤差） |
| 3% | −2.4%（許容範囲） |
| 5% | −4.0%（やや痛いが回復可能） |
| 10% | −8.0%（心理的にもダメージ大） |
| 20% | −16.0%（資産形成が大きく後退） |

「全損しても人生計画が狂わない金額」が上限の目安です。

### 暗号資産投資における日本の税制（超重要）

日本で暗号資産を売却して利益が出た場合、**雑所得**として総合課税されます。これは株式投資の税制と大きく異なります。

| 項目 | 株式・投資信託（分離課税） | 暗号資産（総合課税） |
|---|---|---|
| 税率 | **一律20.315%** | **最大55%**（所得税45%＋住民税10%） |
| 損益通算 | 他の株式等と通算可能 | 暗号資産同士のみ（株式とは不可） |
| 繰越控除 | 3年間可能 | **なし** |
| 確定申告 | 特定口座なら不要の場合あり | 利益20万円超で**必要** |

年収400万円の会社員が暗号資産で100万円の利益を得た場合、所得税率が**20%→23%**の区分に上がる可能性があり、手取りは株式投資で得た場合と比べて**数万円〜十数万円少なくなる**ことがあります。

> ⚠️ **暗号資産の税制は株式に比べて著しく不利**です。この税制面のハンディキャップを上回るリターンが見込めるかどうかが、投資判断の重要なポイントです。

### 金（ゴールド）との比較 — 同じ「代替資産」でもこんなに違う

「インフレヘッジ」「有事の資産」として語られる金とビットコインを比較してみましょう。

| 比較項目 | 金（ゴールド） | ビットコイン |
|---|---|---|
| 歴史 | 数千年の実績 | 約17年 |
| ボラティリティ | 中程度（年率15〜20%） | 極めて高い（年率60〜80%） |
| インカムゲイン | なし（利子・配当なし） | なし |
| 保管コスト | ETFで年0.2〜0.5%程度 | 取引所手数料＋送金手数料 |
| 税制（日本） | 譲渡所得（5年超で長期の優遇あり） | 雑所得（総合課税） |
| 金融危機時の挙動 | 上昇しやすい（逃避先） | 下落することも多い |
| 中央銀行の保有 | あり（信認の裏付け） | なし |

資産防衛を目的とするなら金の方が実績があり、税制面でも有利です。一方、**大きなキャピタルゲインを狙う**なら暗号資産に優位性がある局面もあります。目的に応じて使い分ける視点が重要です。

## 💡 今日のキーポイント

- **ビットコインのボラティリティは株式の3〜5倍。**「投資」というより「投機に近い」ことを理解したうえで、配分を決める
- **ポートフォリオの1〜5%が現実的な配分上限。**「全損しても人生設計が狂わない額」を超えてはいけない
- **日本の暗号資産税制は極めて不利。** 最大55%の総合課税・損失繰越不可・損益通算制限あり。税引後リターンで考える習慣を
- **「分散効果」を過信しない。** 暗号資産は金融危機時に株式と同時に下落する傾向がある
- **コア・サテライト戦略が基本。** まずNISA・iDeCoでコア資産を固め、余裕があればサテライトとして検討する順番を守る

## 🧮 実践例・計算例

### ケース：26歳・年収400万円の会社員が暗号資産をポートフォリオに加える場合

**前提条件：**
- 貯蓄：120万円（うち緊急予備資金として50万円は預金で確保済み）
- 投資可能資金：70万円（新NISAで全世界株式に積立中）
- 毎月の投資可能額：3万円

#### ステップ1：コア資産を優先する

```
【現状の月3万円の配分】
  新NISAつみたて投資枠（全世界株式）  ：3万円/月
  → まずこれを継続・拡大するのが最優先
```

#### ステップ2：サテライトとして暗号資産を検討

毎月の余裕が追加で出た場合（例えば月1万円の余裕ができた場合）：

```
配分案（月4万円の場合）：
  新NISAつみたて投資枠    ：3.0万円（75%）← コア
  暗号資産（BTC）         ：0.5万円（12.5%）← サテライト
  金ETF（特定口座）       ：0.5万円（12.5%）← サテライト
```

#### ステップ3：税引後リターンのシミュレーション

仮に暗号資産に年間6万円（月5,000円）を積み立て、3年後に価値が2倍になったケースと、NISA内の株式が同じく2倍になったケースを比較します。

```
【暗号資産が3年で2倍になった場合】
  投資元本：6万円 × 3年 = 18万円
  評価額  ：36万円
  利益    ：18万円

  税金（雑所得として総合課税）：
    年収400万円 + 雑所得18万円 = 課税所得が増加
    所得税率 10%〜20% + 住民税10% → 概算で約4〜5万円の税負担
    手取り利益：約13〜14万円

【NISA内の株式が3年で2倍になった場合】
  投資元本：18万円
  評価額  ：36万円
  利益    ：18万円

  税金：0円（NISA非課税）
  手取り利益：18万円
```

> 📊 **同じ2倍のリターンでも、手取りで約4〜5万円の差**が出ます。暗号資産が税制上不利であることが具体的な数字で分かります。NISAの非課税枠を使い切る前に暗号資産に資金を振り向けるのは、多くの場合合理的ではありません。

## ✅ 今日のアクションアイテム

### 1. 自分の「コア資産」の充実度を確認する
暗号資産に手を出す前に、NISAの年間投資枠（つみたて投資枠120万円＋成長投資枠240万円）を十分に活用できているか確認しましょう。枠が余っているなら、まずコア資産の充実が先です。

### 2. 「最悪ゼロになってもいい金額」を具体的に書き出す
暗号資産に投資するなら、自分の貯蓄・収入・生活費から逆算して、「この金額なら消えても生活に影響ゼロ」と言える上限を紙に書いてください。感覚ではなく数字で判断する習慣が重要です。

### 3. 暗号資産の税金計算ツールを調べておく
もし暗号資産取引をするなら、Cryptact（クリプタクト）やGtaxなどの**損益計算ツール**の存在を知っておきましょう。確定申告時に計算が複雑になるため、事前準備が必要です。

### Q1. ビットコインの年率ボラティリティは、先進国株式と比べておよそ何倍程度か？

<details>
<summary>答えを見る</summary>

**約3〜5倍**。先進国株式が年率15〜20%程度なのに対し、ビットコインは60〜80%程度のボラティリティがあります。このため、同じ金額を投資しても資産全体への影響度が大きく異なります。
</details>

### Q2. 日本で暗号資産の売却益に課される税金の分類は何か？また、株式の分離課税（約20%）と比べてどう違うか？

<details>
<summary>答えを見る</summary>

暗号資産の売却益は**雑所得**に分類され、**総合課税**の対象となります。税率は所得に応じて最大**55%**（所得税45%＋住民税10%）にもなります。さらに、株式のように**損失の3年間繰越控除ができない**、**株式等との損益通算ができない**という不利な点があります。
</details>

### Q3. ポートフォリオにおける暗号資産の推奨配分比率は一般的にどの程度とされているか？その理由は？

<details>
<summary>答えを見る</summary>

一般的に**ポートフォリオ全体の1〜5%以内**が推奨されています。理由は、仮にビットコインが80%暴落した場合でも、5%配分なら全体への影響は−4%に収まり、資産形成計画を大きく損なわないためです。「全損しても人生設計が狂わない金額」が原則です。
</details>
